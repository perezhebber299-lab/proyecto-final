# proyecto-final
**CHATARRAS EL PANZON** es un sistema de gestión para una tienda de electrónicos hecho en Python. Permite administrar productos, clientes, ventas e inventario desde una interfaz gráfica, y guarda toda la información en una base de datos real en Supabase para que los datos no se pierdan al cerrar el programa. También genera reportes y gráficos de ventas para analizar el negocio.
